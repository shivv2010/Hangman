# Hangman Game 🎮

A fun and interactive Hangman game built using **HTML**, **CSS**, and **JavaScript**. This project is designed to test your guessing skills by finding the hidden word before the man gets hanged!

## 📌 Features
- 🎨 Simple and intuitive UI
- ⌨️ On-screen keyboard for guessing letters
- ☠️ Hangman graphic updates with wrong guesses
- 🔁 "Play Again" functionality
- 🧠 Multiple word options for replay value

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)

## 📂 Project Structure

```
Hangman/
├── assets/             # Images and fonts used
├── index.html          # Main HTML page
├── style.css           # Styling for the UI
└── script.js           # Core game logic
```

## 💡 How to Play

1. A hidden word is chosen at random.
2. Use the on-screen keyboard to guess letters.
3. Each wrong guess draws a part of the hangman.
4. Win by guessing the full word before the man is hanged!
